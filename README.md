# 👟 Thala7 Shoes Store

A modern and responsive shoe store website built using HTML and CSS. The project showcases a complete e-commerce front-end experience including a landing page, product details page, shopping cart, and checkout page.

##Dem0 Links##
link : https://shvam-kumar.github.io/Thala7-shoes-store/index.html

## 🚀 Features

### 🏠 Home Page

* Responsive navigation bar
* Hero section with promotional content
* Customer reviews and ratings section
* Featured products showcase
* About Us section
* Contact Us section

### 👟 Product Details Page

* Large product preview image
* Product description and specifications
* Pricing and discount information
* Quantity selector
* Add to Cart button
* Scrollable image gallery

### 🛒 Shopping Cart Page

* Product listing table
* Quantity controls
* Coupon code section
* Order summary
* Total bill calculation

### 💳 Checkout Page

* Customer email input
* Multiple payment method options
* Card details form
* Country and ZIP code fields
* Order summary card
* Coupon application feature
* Payment provider logos

## 🛠️ Technologies Used

## 🔧 Installation

1. Clone the repository

```bash
git clone https://github.com/your-username/Thala7-Shoes.git
```

2. Open the project folder

```bash
cd Thala7-Shoes
```

3. Run the project

Open `index.html` in your browser.
